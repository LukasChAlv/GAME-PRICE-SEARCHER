# GAME-PRICE-SEARCHER

A Python script that searches for the best video game deals using the CheapShark API and optionally saves them to a CSV file.

## Features

- Search for video game prices and deals via the CheapShark API.
- Optionally save results to a CSV file for future reference.
- Simple and easy-to-use Python command line script.

## Requirements

- Python 3.x

## Installation

1. Clone this repository:
    ```bash
    git clone https://github.com/LukasChAlv/GAME-PRICE-SEARCHER.git
    ```
2. (Optional) Create and activate a virtual environment:
    ```bash
    python -m venv venv
    source venv/bin/activate    # For Linux/Mac
    venv\Scripts\activate       # For Windows
    ```
3. Install necessary dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

1. Run the main script:
    ```bash
    python game_price_searcher.py
    ```
2. Enter the name of the video game you want to search for when prompted.
3. Optionally, save the results to a CSV file.

## Customization

You can modify the script parameters to change the store, price range, and other settings. Refer to the CheapShark API documentation for more information.
